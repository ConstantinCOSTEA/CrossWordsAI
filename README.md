# CrossWordsAI

## Objectifs

L’application mobile doit permettre de photographier une grille de mots croisés depuis un journal, puis de comprendre sa structure (taille, cases noires, définitions, etc.).
Les informations extraites sont ensuite envoyées à un backend IA qui résout la grille, avant de renvoyer la solution au client Android, laquelle est intégrée visuellement dans la photo d’origine.

## Technologies utilisées

Kotlin - Application Android principale

https://ktor.io - Framework serveur pour le back-end

https://docs.koog.ai - Gestion et orchestration du LLM et des clients

WebSocket - Envoi progressif des mots résolus au client
